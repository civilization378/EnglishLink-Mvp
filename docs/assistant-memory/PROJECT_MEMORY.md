# English Link · 项目长期记忆

## 当前项目定位

English Link 是一个微信小程序 MVP，核心目标是通过真实英文场景视频，帮助用户建立“英文 → 场景 → 直接理解”的学习路径，而不是传统翻译式学习。

当前阶段重点不是继续大幅打磨 UI，而是稳定跑通真实学习闭环、补充高质量视频内容、建立视频素材工作流，并准备小范围用户测试。

## Hermes 工作原则

Hermes 默认只分析，不修改。

除非用户明确要求，否则：

- 不修改文件
- 不删除文件
- 不安装依赖
- 不执行 git
- 不读取密钥
- 不大范围扫描
- 不重构项目结构

如果需要修改，必须先说明：

1. 目标
2. 要读取哪些文件
3. 要修改哪些文件
4. 风险点
5. 测试方式

等待用户确认后再执行。
